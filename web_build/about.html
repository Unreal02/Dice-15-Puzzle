기호 표기법

Input 옵션이 Normal인 경우를 기준으로 합니다.

  • U: 위로 이동
  • D: 아래로 이동
  • L: 왼쪽으로 이동
  • R: 오른쪽으로 이동

유용한 공식들

윗면 180도 돌리기

11 블록을 180도 돌립니다.

(D R U L) x 3 / (D L U R) x 3 / (R D L U) x 3 / (L D R U) x 3

윗면과 아랫면 뒤집기 (1)

11 블록의 윗면과 아랫면을 뒤집고 180도 돌립니다.

(D R U L) x 3 / (L D R U) x 3 / (R D L U) x 3 / (D L U R) x 3

윗면과 아랫면 뒤집기 (2)

11 블록의 윗면과 아랫면을 뒤집습니다.

(D R U L) x 3 / (D L U R) x 3 / (D R U L) x 3 / (L D R U) x 3 / (D R U L) x 3

옆면으로 돌리기

15 블록을 다른 방향으로 90도씩 2번 돌립니다.

(D R R U L L) x 2 / D R R U L D L U / (R R D L L U) x 2 / R D L U
